Static Site with Dynamic Data Loading

A demonstration of dynamic data loading from GitHub releases in a static site

View Demo View on GitHub

About

This project showcases how to build a static website that can load dynamic data from GitHub releases. The site is automatically deployed using GitHub Actions, which also generates and publishes new data releases.

Features

  • Static site hosted on GitHub Pages
  • Dynamic data loading from GitHub releases
  • Automated data generation and deployment pipeline
  • Sample data generation with 1000 random entries
