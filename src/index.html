Comments

Want to add your own comment? You can contribute by:

  1. Editing the comments.toml file
  2. Adding your comment using TOML format with markdown support
  3. Creating a pull request

Example format:

[[comments]]
author = "Your Name"
# avatar = "https://your-avatar-url.com/image.jpg"  # You can uncomment this line to use a custom avatar
text = """
# Your Title
Your markdown-formatted comment here
"""
